Writing the {{ SITE_OS_SHORT_NAME }} {{ SITE_OS_VERSION }} ISO file to a USB disk

  1. You will want to find a USB disk with at least 16 GB of storage.
  2. Plug the USB disk into your computer.

It depends on your operating system how you will write the {{ SITE_OS_SHORT_NAME }} {{ SITE_OS_VERSION }} ISO to a USB disk. Here are some instructions for creating a {{ SITE_OS_SHORT_NAME }} {{ SITE_OS_VERSION }} USB disk if you have a computer running Microsoft Windows or Linux.

If you are on Microsoft Windows, I recommend downloading a program called Rufus to write the ISO to a USB disk.

  1. Download the Rufus application at https://rufus.ie/en/.
  2. Find the latest .exe download for platform "Windows x64".
  3. Write the {{ SITE_OS_SHORT_NAME }} {{ SITE_OS_VERSION }} ISO to a USB disk with the Rufus application.

Linux instructions for writing ISO files to a USB disk

If you have a Linux Gnome desktop environment, you can use the Disks application to write an ISO to a USB disk.

  1. Open the Disks application.
  1. Plug in your USB disk into your computer, and watch for your USB disk appear in the Disks application. Make sure you have selected the correct USB disk, because the next step will require root privileges and destroy any existing files on the disk you select.
  1. Once you have selected the correct USB disk, click the button with 3 vertical dots at the top right of the Disks application.
  2. Select Restore disk image….
  1. Click on the Image to Restore button.
  2. Select Restore disk image….
  1. Select the {{ SITE_OS_SHORT_NAME }} {{ SITE_OS_VERSION }} ISO file {{ SITE_OS_SHORT_NAME | lower }}-{{ SITE_OS_VERSION }}-x86_64-dvd.iso in your Downloads.
  2. Press the Open button.
  1. Click the Start Restoring… button to start writing the {{ SITE_OS_SHORT_NAME }} {{ SITE_OS_VERSION }} ISO to the disk.
  1. Another window will open for you to verify that the USB disk you wish to format is selected by looking closely at the size of the disk or it's name so you don't format the wrong disk.
  1. Another window will open for you to authenticate with your user password as an administrator, which requires root privileges to write to the disk.
  2. Enter your user password.
  3. Press the Authenticate button to begin writing the {{ SITE_OS_SHORT_NAME }} {{ SITE_OS_VERSION }} ISO to the USB disk.
  1. Wait patiently for a few minutes for the disk to finish writing to the disk.
