In this course:

  1. Learn how to deploy and manage a Hashicorp Vault for managing and sharing serets with applications securely.
  2. Learn how to deploy cert-manager and manage TLS/SSL certificates for Keycloak or other applications for secure connections between your applications.
  3. Learn how to deploy Apache Zookeeper cluster manager for scaling your applications across multiple computers for high availability.
  4. Learn how to deploy Apache Solr for building searchable applications for users to easily find what they are looking for.
  5. Learn how to deploy Crunchy Postgres Operator for managing SQL relational databases for data-driven appications.
  6. Learn how to deploy Keycloak Single Sign-On server, providing zero-trust fine-grained access control to your data-driven applications.
  7. Learn how to deploy FIWARE's ScorpioBroker Context Broker, for managing IoT edge device data in the cloud.
