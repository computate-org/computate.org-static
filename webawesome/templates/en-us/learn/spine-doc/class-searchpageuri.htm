Web pages can be defined for searching the data model as a search page. allowing users with access to a data model to search those data models. so SPINE Programming makes it easy to define web pages in addition to APIs to perform common REST (Representational State Transfer) verbs to POST, PATCH, GET, DELETE, and PUT data into a database and search engine, and retrieve model instances as JSON data. The {{ result.commentVar }} is used as the base URI for search page of the Java class data model.

Search pages in multiple languages are supported in SPINE Programming as an ApiMethod:. This involves specifying a unique ID SearchPageFrFR:, for the data model in the given language. This also involves specifying the Language: frFR, where the fr stands for the language "français", and the FR stands for the country "France" where the language is spoken locally. This also includes the Java class Page: FishPopulationPage that is used to render the page in that language. The ApiUri: /fr-fr/rechercher/population-poissons is a unique API URI separate from other languages where the page can be served in that language.
