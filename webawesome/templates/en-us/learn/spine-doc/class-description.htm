Tracking fish population counts, maturation, and incubation over time.

The Computate SPINE Programming platform at this time supports code generation in French and English. The entire Computate code base is entirely written in French, and primarily builds English websites as a second language through internationalization files with YAML translations in French and English. This requires adding each locale ( .enUS, .frFR ) after the Description, but before the :. 

/**
 * Description.enUS: Tracking fish population counts, maturation, and incubation over time. 
 * Description.frFR: Suivi du nombre de poissons, de leur maturation et de leur incubation au fil du temps. 
 */
public class FishPopulation extends FishPopulationGen<BaseModel> {
}

It has also been used to generate French and English code at the same time, but this is not always a requirement of every website, but is available for those use cases.
