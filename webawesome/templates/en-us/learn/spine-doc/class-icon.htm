As a form of engaging visual communication, you can also define visual cues for identifying a data model in the user interface by defining an icon from an icon library. Above we define a fish icon from the Font Awesome library to the {{ classSimpleName }} data model.

When the user sees the fish icon, it serves as a visual cue that reminds the user that it's related to {{ classSimpleName }}.

The Icon contains an environment variable {{ '{{ ' + 'FONTAWESOME_STYLE' + ' }}' | replace("{{", "{{") | replace("}}", "}}") }} inside double curly braces. The reason for this is to allow for a consistent, flexible choice of Font Awesome Icon Packs at runtime. Font Awesome provides many fun Icon Packs to customize the look and feel of your website. Your data models extend this Font Awesome flexibility if you include this {{ '{{ ' + 'FONTAWESOME_STYLE' + ' }}' | replace("{{", "{{") | replace("}}", "}}") }} variable in your model icon.
