What is the {{ result.name }}?

Teaching tools and automation for easily building a custom website with the models and data that you care about using the best in open source software.

Modern development tools

Use VSCode and interactive Jupyter notebooks to start writing code like a real cloud-native developer.

Cloud-native microservices

Deploy cloud microservices in a Red Hat OpenShift Local cloud environment on your own computer for certificate management, event bus application clustering, databases, a search engine, and single sign-on server for securing and scaling your data-driven website.

Automate tasks

Use Ansible automation to generate all of the boilerplate code for your Smart Aquaculture site, and customize it along the way.

Learn reactive programming

Learn how to build reactive asynchronous Java code, OpenAPIs, database scripts, and Smart Aquaculture data.

Build interactive dashboards

Visualize your Smart Aquaculture data in event-driven, interactive dashboards and maps.

Cloud-native microservices

Interact with modern microservices in the cloud like:
  1. PostgreSQL Database for transactional data storage
  2. Apache Zookeeper Cluster Manager for event bus messaging
  3. Apache Solr Search Engine for autosuggest and searchable APIs
  4. Keycloak Single Sign-On for fine-grain access control to Smart Aquaculture data
  5. FIWARE Context Broker for open source IoT edge device standards
