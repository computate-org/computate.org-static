What is the {{ result.name }}?

Learn how to develop custom Smart Aquaculture data-driven websites and open APIs in the cloud, and build your own secure IoT smart platform with generative AI, zero-trust access control, and interactive dashboards.

The Smart Aquaculture project, a collaboration between Green Reef Foundation, Oceans 97, and Qatar Strategic Business Group, supports food processing in developing nations through workforce training with academic and industry partners using innovative digital twins, unmanned aerial vehicle (UAVs), and autonomous aerial vehicles (AAVs). Arise Fish and Seafood Process and the Green Reef Foundation combine high-fidelity simulations with digital culture preservation and training on open source edge computing to support these digital twins.

In this hands-on lab, we’ll develop new smart aquaculture data models to run on an intelligent data platform, powered by the open source FIWARE framework and a Cybersecurity Special Library Operations framework of Red Hat-defined edge solutions. With this platform, we’ll import and visualize fish and population data in a Red Hat OpenShift AI Visual Studio Code (VSCode) Java workbench to help mitigate food insecurity. You’ll gain experience using edge device data within a hybrid cloud environment, including:

  1. Intelligent application programming interface (API) and dashboard code generation with the Computate platform.
  2. Smart device registration with FIWARE microservices.
  3. Message data transmission to Red Hat AMQ Broker.
  4. Event-driven subscriptions to smart device data changes.
  5. Granular data access control between edge and cloud with Red Hat build of Keycloak.
  6. Custom Leaflet dashboards for visualizing live GeoJSON data on maps.

Modern development tools

Access your VSCode developer workbench in the browser, it doesn't matter your laptop's operating system.

Automate tasks

Use Ansible automation to generate all of the boilerplate code for your Smart Aquaculture site, and customize it along the way.

Learn reactive programming

Learn how to build reactive asynchronous Java code, OpenAPIs, database scripts, and Smart Aquaculture data.

Build interactive dashboards

Visualize your Smart Aquaculture data in event-driven, interactive dashboards and maps.

Cloud-native microservices

Interact with modern microservices in the cloud like:
  • PostgreSQL Database for transactional data storage
  • Apache Zookeeper Cluster Manager for event bus messaging
  • Apache Solr Search Engine for autosuggest and searchable APIs
  • Keycloak Single Sign-On for fine-grain access control to Smart Aquaculture data
  • FIWARE Context Broker for open source IoT edge device standards
