What is the {{ name }}?

Learn how to develop custom Smart Aquaculture data-driven websites and open APIs in the cloud, and build your own secure IoT smart platform with generative AI, zero-trust access control, and interactive dashboards.

Modern development tools

Access your VSCode developer workbench in the browser, it doesn't matter your laptop's operating system.

Automate tasks

Use Ansible automation to generate all of the boilerplate code for your Smart Aquaculture site, and customize it along the way.

Learn reactive programming

Learn how to build reactive asynchronous Java code, OpenAPIs, database scripts, and Smart Aquaculture data.

Build interactive dashboards

Visualize your Smart Aquaculture data in event-driven, interactive dashboards and maps.

Cloud-native microservices

Interact with modern microservices in the cloud like:
  • PostgreSQL Database for transactional data storage
  • Apache Zookeeper Cluster Manager for event bus messaging
  • Apache Solr Search Engine for autosuggest and searchable APIs
  • Keycloak Single Sign-On for fine-grain access control to Smart Aquaculture data
  • FIWARE Context Broker for open source IoT edge device standards
